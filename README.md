# Receipts
by Phil Mass

Deployed at [https://philrmass.github.io/receipts/](https://philrmass.github.io/receipts/)

# Setup
```
git clone git@github.com:philrmass/receipts.git
cd receipts
yarn
```

# Dev Server
```
yarn dev
open http://localhost:8080
```

# Production Build & Deploy
```
yarn deploy
```
